| الشطر الأول | الشطر الثاني |
| :-- | :-- |
|  بلا مودِّنة عارف زواجل طيري   |  والحمد لله اللى بسط جرتّا  |
| ولا انظري ناسيهن اصوات هديري | فاوقات الانظار اللى انذهب شيرتّا |
| يا صاحبي السيّه انه تفكيري | ما جابها البيعة وسهرايتّا |
| وراي ؟ لاااا لقيتك فيّ نافخ كيري | ضلن جواجيك هذي حجتّا ؟؟ |
| تربح ولا لكيفك ينوض طفيري | ولا تصيم عيني عن اللى ولاتّا |
| بيك وبلاك وبيّ نا وبغيري | تطّايب اجروحي تنقتل غصتّا |
| امطارهن ارعودي مكفيات شعيري | علي قلها ما اسمح طعم غلتّا |
| يحاذر خبيري من ادروب خبيري | بعيون ما تلاصيف البشر غرتّا |
| ولا راس مال إلا امفيت ضميري | وروح تخشى يوم تلظايتّا |
| رايمه المعاني اللافيه لغويري | اللى سيوف سافيها صلت غطتّا |
| ولا ب هدهد اسليمان فى تفسيري | جبتا نبا بلقيس واحكايتّا |
| ان مو جبح عقلي مرمّزه تشفيري | فى صمتّا تبليغ ل كليمتّا |
| جايني كما اركيّب نوا تدميري | قليده امحرم نطق رجّابتّا |
| لا انباع لا لدينار لا للميري | ولا غرّ به تصفيق لشرايتّا |
| قصيرة اقيون وختمها تقديري | سلّم اعيوني هي اللى ربتّا |
| بنات عقل حاضيهن دفا كشميري | ما اجرود ((فبراير)) اللى كستّا |
| ولو الشعر ترجمهن لغاوي بيري | ما رُمت قافيته ولا انغمتّا |
