---

| الشطر الأول | الشطر الثاني |
| :--- | :--- |
| رِيــــتــه وتـــو اتــحــقها قــانــصّه |وتقول كــنــت انــريــدها إلا لــوّ |
| مــا جــابها اخريفيش صاعب نصّه | يــنــوّض الــحرشا اتــقول هــيا تــوّ |
| مــواريث جــده وهو عليها موصّى | ولا خــير فــيه ان عاف مــا سمّو |
| وتــابــع ادروبـــه كــلها مــاصصّه | هــضاك كــان هــو وهــضا هو هوّ |
| مـــا قــالها مــن قــبل مــا شــخصّه | كــي عــيت ( حــيها ) لمّدو وحشوّ |
| وراك تــعرفه قصدي وهذيك القصّه | ومــن هو اللي جفّل ومن ( شاكوّ ) |
| وزاد تــعرفه مــن هــو اللي نغصّه | زمــرة اكــلاب القاف عيت ( الشوّ) |
| يــجو كــلمتين ويــغلبو ف الــرصّه | مــاريــت فــيــهن م الــمــواير قــوّ |
| ولــو الاصــمعي هضاك مستخلصّه | خــبــرة تــجــارب مـــن زمــانه أوّ |
| عــــقــل رــــبي خــالــقه يــقــلصّه | فـــي شــطر شــعري او بــما أبــدَوّ |
| انـــه يــمتلك اعــين فــقط تــمتصّه | مـــن ســمح مــا قــالو ومــا خــطَوّ |
| انــا أؤكــد لــك بــانك لــو اتمحصّه | ســاتــقول لـــي بــالــك بــها جــلَّوّ |
| وانـــه شــبيه الــماس نــادر فصّه | خــافي عــلي دوج الــعرب والضوّ |
| ومـــا تــرجمته بــلتاجي امــخلصّه | مـــن كـــل شــائبة تــحتوي الــسوّ |
| ولــو كــان مــا تعرف انه مخمصّه | وانـــه قــايــل ووصــلتم ومــا ردَوّ |
| الــرمحي ومــا يرجى اللي يرخصّه | شــــعــارهــا ونــقــادهــا والــــجــوّ |
| وحرصا على التوضيح ماهي حصّه | فــي لــغة ضــاد الــلي لــها احــتمَوّ |
| نــردو لــموضوعي الــلي حــمصّه | اكــبــودهم بــما راعــو ومــا ابــتلَوّ |
| بــارماق دامــي ف الوغى مختصّه | عــارفــينها جــتــهم بــهــا انـــرزوّ |
| واضــع بــها دســتورها امــخصصّه | مـــا يــرومــها الا مَــن لــها ادّعــوّ |
