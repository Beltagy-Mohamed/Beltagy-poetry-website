| الشطر الأول | الشطر الثاني |
| :--- | :--- |
| يا نجع ما هوّنت راي لشايب | ويا نجع ما كبّرت يوم احوار |
| تبويعتك لهلة كل لافي غايب | تشهد عليها كل رمة دار |
| يا نجع تذبح م اللذيذ الطايب | يا نجع لا ما حذاك جايع جار |
| يا نجع عن فعل الرديه تايب | يا نجع بسماح الفعايل بار |
| يا نجع لمة هلك فيك اتهايب | في كل ذيل ايريد فيك دمار |
| يا نجع لا كشّرت قول كتايب | عالنه احروب وقتل واستنفار |
| يا نجع تهلب في كلام الخايب | يا نجع لا ما حاد بيك مسار |
| يا نجع لا ام التوم لا ام جنايب | تصبح غفود ولا خلوج عشار |
| يا نجع جود الجود منك جايب | سيمته علي مر الازمان اخيار |
| يا نجع خلّك من زفيّ العايب | القايل البوم انه سليم مطار |
| مشايخ ابطون وف العقول ذهايب | يحاحو لقطا قالو طيور حرار |
| وياما احملنا م الزمان غرايب | برم وقتنا والحبل ع الجرار |
| يا نجع ديما كيف لون الرايب | جهيمك اتقول انه عقاب اغرار |
