| الصدر | العجز |
| :--- | :--- |
| والله الكلام لو علّا ملل | ما الاعمال المولى حطّها |
| وسمح الكلام ما دل وكمل | سهمه مدّ لو طال المدا |
| مراده سمح داوا لو هطلّ | ودوم السر صدرا كامرا |
| سلم م المكر واعكاس العلل | عدو الكرى والاسر والمحوا |
| لاح الود مسلول طالل | طواه مطر مطاول طما |
| كاسه ملك راسه مكلل | ماسح كِل ما دار الصدا |
| وصام الهوا الدهر كامل | هول السَّحَر كله رعـا |
| ولم العلم صال حامل | سهم العمر صار مرسا |
| مدحه الهمّ حلّاه حوّل | عطره مهر طواها الوطا |
| ولو راد الهلال هوّل | ما هل لا حل حلّا |
| وسال الهلال هل هل ؟ | ولو هل الهلال كملّا |
| طال هود دعا عاد ما ملّ | مو مُهِِم وصّل الدعوا |
| المُهِم الراى عدّل | عمل صح وما صح لولا |
| ولا طاح الا ووصل | اعلومه المُرسل لها |
| وكاد كِل حسّاد هامل | سود المدامع ردمها |
| كلام رمح لا صاح صاهل | ولد ارماح احرار الدما |
| دردر اكرومر وململ | عدو دار الاهرام كدّا |
| صده حلم ملوك الدول | سرا السر كسواه الرحا |
| حمد رد لمصر حامل | حصاد عمر ألماس مالطا |
