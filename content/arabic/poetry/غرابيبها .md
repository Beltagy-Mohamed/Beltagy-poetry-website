| الشطر الأول | الشطر الثاني |
| :--- | :--- |
| غرابيبها حاين سجايا ضيّ | ضميره محل إعراب كلمة لنّ |
| شواطيه غرّن به اتبّع غيّ | مبادي اتكسي روحها بكفنّ |
| عليه عالنين اسيوف عيت الكيّ | امغير سيفهم حدّه يريد مسنّ |
| كيفه الدوج لهن صليل رديّ | بعَد نوضته ف الناس قالو غَنّ |
| واسلمت وعمّر اعقيلك بَيّ | وواهمين جد البَيّ كاين منّ ؟ |
| وما اتقول لي لنّا افعال الحيّ | ياما ولد سَلول وَدّ وحَنّ |
| وهو فيه يتّقه بقصر الديّ | ميّس انفوس اتموت كان صفَنّ |
| وراهو جمال الشيّ مو ف الشيّ | امغير لعقول الذاهبه ما بنّ |
| قولو لهم بو الزغل كان وفيّ | وقولو لغرناطه آمال حسنّ |
| وقولو لهم يرحم الله عليّ | ما هو علي طهران لا سَاسنّ |
| خوذو لغلاف ونا اكتابي ليّ | ما انسلّمه طايع ولا بثمنّ |
| وهضا لاو بشرع ولا كلام نبيّ | ولا هو بجن إن كان قالو جنّ |
